## Overview

POSCLUS is an open-source platform for deploying and managing production-ready PostgreSQL HA clusters using Patroni.

## Features

- **Agent-based Architecture** - Lightweight agents deployed on each node
- **Web Dashboard** - Modern UI for cluster management
- **Patroni Integration** - Battle-tested HA with Patroni + etcd
- **Backup Support** - pgBackRest integration for reliable backups
- **Real-time Monitoring** - Live metrics via SignalR
